# PLANORA

PLANORA is a personal planner web application built using HTML, CSS, and JavaScript. It helps users manage daily tasks, track progress, and maintain a private diary in one place.

## Features

- User login screen
- Task creation and management
- Task search option
- Priority levels (high, medium, low)
- Progress tracker
- Weather information display
- Motivational quotes
- Private diary with password protection
- Local storage support
- Theme toggle option

## Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome
- Open Meteo API
- Quotable API

## How to Run

1. Download or clone the project files
2. Open the project folder in VS Code
3. Open `plan html.html` in browser
4. Start using the application

## Project Purpose

This project is designed to improve personal productivity by organizing tasks and notes in a simple and interactive interface.
